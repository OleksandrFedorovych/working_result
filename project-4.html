These tools are custom automations I built for various use cases including Reddit scraping, LinkedIn profile enrichment, Google Sheets integration, Zoho Books invoicing, and more. Technologies used include n8n, Airtable, Supabase, Google Apps Script, GitHub Actions and custom JS code.

3. Google Sheet ↔ Webform Integration:

Using Google Apps Script, this automation imports data from a sheet into a webform (including PNG/PDF attachments) and also generates downloadable PDF files.

4. LinkedIn Scraper:

Custom JS + Supabase backend that scrapes structured LinkedIn profile data automatically from user-inputted URLs on a frontend form.

5. Pickles Car Scraper via GitHub Actions:

Every 10 minutes, GitHub Action checks for car listing updates using filters, saves new data to Supabase, and alerts clients on updates.

6. Zoho Books & Google Sheets Integration:

Google Apps Script automatically creates Zoho invoices based on Google Sheet data.
